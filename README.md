## 🚀 Installation

1. Clone this repository:
```bash
git clone https://github.com/NekoClub/Mine-4-Me.git
cd Mine-4-Me
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure your settings in `config.json`:
```json
{
  "goddess_wallet": "bc1q_your_goddess_wallet_here",
  "mining_intensity": "medium",
  "auto_start": false,
  "show_teases": true,
  "tease_frequency": 300,
  "transparency_mode": true,
  "donation_percentage": 100,
  "mining_algorithm": "randomx"
}
```

## ⚙️ Configuration Options

| Option | Description | Default |
|--------|-------------|---------|
| `goddess_wallet` | Cryptocurrency wallet address for Goddess | Required |
| `mining_intensity` | Low, medium, or high CPU usage | medium |
| `auto_start` | Start mining automatically on launch | false |
| `show_teases` | Display teasing messages during mining | true |
| `tease_frequency` | Seconds between tease messages | 300 |
| `transparency_mode` | Show full transparency info at startup | true |
| `donation_percentage` | Percentage of mining to Goddess (always 100) | 100 |
| `mining_algorithm` | Mining algorithm to use | randomx |
